# Framework Comitê de Investimentos para GPT

Copyright © 2026 Marcus Vinicius Cirino e Silva.

Este repositório contém a documentação oficial do **Framework Comitê de Investimentos para GPT**.

O Framework foi desenvolvido para apoiar a utilização de Grandes Modelos de Linguagem (LLMs) como Comitês de Investimentos voltados à gestão patrimonial de investidores pessoa física brasileiros.

O projeto reúne:

- Framework metodológico;
- Base de Conhecimento (Knowledge);
- Protocolos operacionais;
- Documentação de apoio;
- Modelos para configuração de GPTs personalizados.

## Finalidade

Este projeto possui finalidade exclusivamente educacional e metodológica.

Seu objetivo é disponibilizar uma estrutura organizada para utilização de Inteligência Artificial como ferramenta de apoio à tomada de decisão patrimonial.

Este Framework não constitui:

- recomendação de investimentos;
- consultoria financeira;
- consultoria tributária;
- consultoria jurídica.

Todas as decisões de investimento permanecem sob responsabilidade exclusiva do investidor.
